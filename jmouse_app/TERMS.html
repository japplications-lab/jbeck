Purpose & Functionality

jMouse prevents system idle states by simulating keyboard input (F15 key) and subtle mouse movements at randomized intervals. This keeps your workstation active during extended operations, remote sessions, or long-running tasks.

System Integration

Windows Registry

jMouse may write to your user-level Windows registry (HKCU) to enable automatic startup. This does not require administrator privileges and can be disabled at any time.

Background Operation

When minimized to system tray, jMouse continues running. A tray icon indicates active status.

Persistent Settings

Your preferences are stored locally in jmouse_settings.json. This file persists after closing the app.

Input Monitoring

Global Hotkeys

jMouse monitors keyboard input for hotkeys (Ctrl+Alt+E/M/Q). No keystrokes are recorded, stored, or transmitted. Monitoring stops when the app closes.

Idle Detection

jMouse monitors mouse and keyboard activity timing to detect idle states. Only timestamps are tracked‚Äîno content is recorded or logged. Data is never stored or transmitted.

Automated Operation

Scheduled Automation

If configured, jMouse can automatically start/stop based on your weekly schedule. This is opt-in and can be manually overridden at any time.

Detection & Monitoring

While jMouse uses randomization (45-75 second intervals, variable movement patterns) to appear more human, advanced monitoring software may still detect:

  • The jMouse.exe process running on your system
  • Patterns in automated input behavior
  • Registry entries (if Windows startup is enabled)
  • Keyboard/mouse activity monitoring hooks

Bottom Line: jMouse reduces but does not eliminate detection risk. Use ethically and in accordance with workplace policies.

Privacy & Data

What jMouse Monitors

  • Activity Timing: Timestamps of mouse/keyboard events (for idle detection)
  • Hotkey Combinations: Detection of Ctrl+Alt+E/M/Q presses

What jMouse Does NOT Do

  • Record keystrokes or mouse clicks
  • Log browsing history or application usage
  • Transmit any data over the internet
  • Store activity history
  • Access personal files or documents

All processing is local. jMouse has no network connectivity and does not phone home.
