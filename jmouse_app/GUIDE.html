üìñ What is jMouse?

jMouse is a lightweight desktop application that prevents your Windows computer from going idle. It accomplishes this by:

  • Simulating subtle mouse movements (¬±5 pixels) every 60 seconds
  • Pressing the F15 key (a virtual key that doesn't interfere with your work)
  • Running quietly in the background or system tray

This keeps your screen from locking, prevents screensavers from activating, and maintains your "active" status in communication apps like Microsoft Teams, Slack, or Zoom.

üí° Intended Use Cases:
  • Long-running downloads or file transfers
  • Remote desktop sessions that time out
  • Monitoring dashboards that need to stay visible
  • Presentations or demos
  • Keeping systems awake during overnight processes

‚è∞ Scheduled End Time

You can configure jMouse to automatically disable itself at a specific time:

  1. Check the "Scheduled End Time" checkbox

üí° Tip: The schedule cannot be changed while jMouse is enabled. Disable it first if you need to adjust the time.

‚öôÔ∏è How It Works

jMouse operates by sending low-level input events to Windows every 60 seconds:

  • Mouse movement: Moves the cursor by a random offset of ¬±5 pixels (barely noticeable)
  • F15 keypress: Simulates pressing the F15 key (a virtual key that has no effect on most applications)

These inputs trick Windows into thinking you're actively using the computer, preventing idle timeouts, screen locks, and sleep mode.

‚ö†Ô∏è Important: jMouse does not prevent you from manually locking your computer (Win+L) or putting it to sleep. It only prevents automatic idle timeouts.

üÜò Troubleshooting

Problem: jMouse doesn't keep Teams/Slack active

Solution: Some apps (like Teams) have their own inactivity detection that may override system-level activity. jMouse works for most cases, but presence detection varies by app.

Problem: Antivirus blocked or quarantined jMouse

Solution: Add jMouse.exe to your antivirus exclusion list. The detection is a false positive due to PyAutoGUI and PyInstaller.

Problem: Windows SmartScreen blocks jMouse

Solution: Click "More info" ‚Üí "Run anyway". The app is not code-signed, which triggers SmartScreen warnings for unsigned executables.

Problem: jMouse stops working after a while

Solution: Check if Windows Defender or your antivirus quarantined the app. Also ensure jMouse wasn't accidentally disabled via the tray icon.
